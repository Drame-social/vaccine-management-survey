# Limitations

1. This project uses synthetic data only. It does not estimate real VFC, CDC, awardee, provider, or patient performance.
2. Public row-level Vaccine Management Survey data were not identified as available for GitHub use.
3. Synthetic waste rates were generated to create realistic analytic variation, not to represent factual vaccine-specific loss rates.
4. Synthetic awardee names, regions, program types, staffing counts, policies, orders, waste, and incident categories are fictional.
5. The dashboard and code demonstrate analytic skill; they should not be used for operational decision-making.
6. The optional standard analysis excluding COVID-19 and influenza is included because the original SAS workflow contained a similar sensitivity-style analysis. It is a portfolio method, not a CDC recommendation.
7. Some synthetic missingness is intentionally introduced to demonstrate QA procedures.
8. Reconciliation discrepancies are intentionally introduced for a subset of awardees to demonstrate data-quality follow-up.
